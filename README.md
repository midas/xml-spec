# XmlSpec

XML matchers for RSpec and Cucumber.

#### This is alpha software and is not yet ready for primte-time.

## Installation

Add this line to your application's Gemfile:

    gem 'xml_spec'

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install xml_spec
